HSGames - Tienda virtual de videojuegos

Información del proyecto

  • Categoría: Desarrollo web
  • Tecnologías: HTML, CSS, JS, Node.js
  • Repositorio: https://github.com/HinaraSM12/HSGames

El proyecto consiste en una tienda virtual de productos relacionados con videojuegos llamada HSGames. La plataforma permite a los usuarios buscar y explorar una variedad de productos, ver detalles de los mismos y, en el futuro, agregar productos al carrito, crear una cuenta y realizar compras.

Las principales funcionalidades implementadas hasta ahora incluyen:

  • Explorar Productos: Los usuarios pueden navegar por la tienda y ver los productos disponibles en diferentes categorías, como videojuegos y consolas.
  • Buscar Productos: Se puede realizar una búsqueda de productos por nombre.
  • Ver Detalles del Producto: Al hacer clic en un producto, se muestran detalles como el nombre, la descripción, el precio y una imagen.
  • Funcionalidades CRUD para Administradores: Los administradores pueden agregar, editar y eliminar productos desde un panel de administración, lo que permite mantener actualizado el catálogo de productos.

Para la implementación, se utilizó HTML, CSS y JavaScript para el frontend. El backend está simulado con JSON Server, una herramienta que permite crear una API REST a partir de un archivo JSON, lo que simplifica el desarrollo y la prueba de la aplicación sin la necesidad de una base de datos real.

En el futuro, se planea integrar funcionalidades adicionales como el carrito de compras, la creación de cuentas de usuario y la funcionalidad de compra para hacer la experiencia de compra más completa y satisfactoria para los usuarios.
